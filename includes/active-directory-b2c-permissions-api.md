1. 選取 [應用程式]  ，然後選取應可存取 API 的 Web 應用程式。 例如，*webapp1*。
1. 選取 [API 存取]  ，然後選取 [新增]  。
1. 在 [選取 API]  下拉式清單中，選取 Web 應用程式應獲得其存取權的 API。 例如，*webapi1*。
1. 在 [選取範圍]  下拉式清單中，選取您先前定義的範圍。 例如 *demo.read* 和 *demo.write*。
1. 選取 [確定]  。
