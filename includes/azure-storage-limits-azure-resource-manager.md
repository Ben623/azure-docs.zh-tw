只有當您使用 Azure Resource Manager 搭配 Azure 儲存體來執行管理作業時，才適用下列限制。

| Resource | 預設限制 |
| --- | --- |
| 儲存體帳戶的管理作業 (讀取) |每 5 分鐘 800 |
| 儲存體帳戶管理作業 (寫入) |每小時 200 |
| 儲存體帳戶的管理作業 (清單) |每 5 分鐘 100 |
