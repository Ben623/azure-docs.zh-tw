1. 在您虛擬網路的 [**設定**] 頁面上, 流覽至 [ **dns 伺服器**], 然後按一下以開啟 [ **dns 伺服器**] 頁面。

    ![指定 DNS 伺服器](./media/vpn-gateway-add-dns-rm-portal-include/dns-server.png "指定 DNS 伺服器")

   - **DNS 伺服器：** 選取 [自訂]。
   - **新增 DNS 伺服器：** 輸入您想要用於名稱解析之 DNS 伺服器的 IP 位址。

2. 完成 DNS 伺服器新增時，按一下頁面頂端的 [儲存]。
