1. 在虛擬網路頁面的 [設定]  區段中，選取 [DNS 伺服器]  以開啟 [DNS 伺服器]  頁面。

2. 在 [DNS 伺服器]  頁面上，填入下列值：

   - **DNS 伺服器**：選取 [自訂]  。

   - **新增 DNS 伺服器**：輸入您想要用於名稱解析的 DNS 伺服器的 IP 位址。

3. 當您完成新增 DNS 伺服器時，選取 [儲存]  。

    ![指定 DNS 伺服器](./media/vpn-gateway-add-dns-rm-portal-include/specify-dns-server.png "指定 DNS 伺服器")
