  | | |
  |---|---|

使用下列命令來安裝所需的 strongSwan 組態：

```
apt-get install strongswan-ikev2 strongswan-plugin-eap-tls
```

```
apt-get install libstrongswan-standard-plugins
```

```
apt-get install strongswan-pki
```
