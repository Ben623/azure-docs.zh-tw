- **訂單詳細資料**。 建立訂單時，寄送地址、 電子郵件地址和使用者的連絡資訊儲存在 Azure 入口網站中。 儲存的資訊包括︰
  - 連絡人姓名
  - 電話號碼
  - 電子郵件地址
  - 街道地址
  - City
  - 郵遞區號/郵遞區號
  - State
  - 國家/地區/市/區
  - 運送追蹤號碼

    訂單詳細資料會經過加密及儲存在服務中。 服務會保留這些資訊，直到您明確地刪除資源或順序。 刪除資源和對應的訂單會封鎖該裝置隨附的裝置傳回給 Microsoft 之前的時間。

- **交貨地址**。 訂單之後，資料箱服務會提供寄送地址至 UPS 等第三方電訊廠商。

- **共用使用者**。 在您的裝置上的使用者也可以存取位於共用上的資料。 您可以檢視的使用者可以存取共用資料的人員清單。 刪除共用，也會刪除這份清單。
