虛擬機器 (VM) 會自動指派的私人 IP 位址範圍從您指定的子網路中部署。 刪除 VM 之前，VM 會保留的位址。 以動態方式，azure 會從您建立的虛擬機器的子網路指派的下一步 可以使用私人 IP 位址。 如果您想要從指派給 VM 的子網路的特定 IP 位址，請指派靜態 IP 位址。
