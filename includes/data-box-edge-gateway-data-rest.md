待用資料:

- 存取儲存在共用中的資料會受到限制。

    - 存取共用資料的 SMB 用戶端需要與共享相關聯的使用者認證。 建立共用時, 會定義這些認證。
    - 建立共用時, 必須新增存取共用之 NFS 用戶端的 IP 位址。
