對於待用資料：

- BitLocker XTS-AES AES 256 位元加密來保護本機資料。
- 資料儲存在共用的存取權會受到限制。

    - 存取共用資料的 SMB 用戶端需要與共用相關聯的使用者認證。 建立共用時，會定義這些認證。
    - 建立共用時，會新增要存取共用的 NFS 用戶端的 IP 位址。
