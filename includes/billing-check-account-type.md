檢查來判斷您的帳單帳戶類型的合約類型
 
1. 登入 [Azure 入口網站]( http://portal.azure.com)。
 
2. 針對 [成本管理 + 帳單]  進行搜尋。
 
   ![顯示 Azure 入口網站搜尋的螢幕擷取畫面](./media/billing-check-account-type/billing-search-cost-management-billing.png)    
 
3. 如果您只有一個帳單寄送的範圍內存取，請選取**屬性**從左上方。
 
    ![在 [屬性] 頁面會顯示 microsoft 客戶合約的螢幕擷取畫面](./media/billing-check-account-type/billing-mca-property.png)
    
    **計費帳戶類型**屬性頁會決定您的帳戶類型。 它可以是 Microsoft 線上服務計劃、 Enterprise 合約或 Microsoft 客戶合約。 若要深入了解的計費帳戶類型，請參閱[Azure 入口網站中檢視您的帳單帳戶](../articles/billing/billing-view-all-accounts.md)。  
 
4. 如果您需要存取多個計費的領域，請選取**帳單帳戶**或**計費的所有領域**從左上方。 檢查的型別**計費帳戶類型**資料行。
 
    ![在 帳單的 帳戶 清單頁面上顯示 microsoft 客戶合約螢幕擷取畫面](./media/billing-check-account-type/billing-account-type-in-the-list.png)
