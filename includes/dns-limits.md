**公用 DNS 區域**

| 資源 | 預設限制 |
| --- | --- |
| 每個訂用帳戶的公用 DNS 區域 |250 <sup>1</sup> |
| 每個公用 DNS 區域的記錄集 |10,000 <sup>1</sup> |
| 每筆記錄的記錄集的公用 DNS 區域中 |20 |
| 每個訂閱的私人 DNS 區域 |1000|
| 每個私人 DNS 區域的記錄集 |25000|
| 設定適用於私人 DNS 區域的每筆記錄的記錄 |20|
| 每個私人 DNS 區域的虛擬網路連結 |1000|
| 啟用自動註冊的私人 DNS 區域每一虛擬網路連結 |100|
| 虛擬網路可以取得連結到已啟用自動註冊的私人 DNS 區域數量 |1|
| 您可以取得連結的虛擬網路的私人 DNS 區域的數目 |1000|

<sup>1</sup>如果您需要增加這些限制，請連絡 Azure 支援服務。
