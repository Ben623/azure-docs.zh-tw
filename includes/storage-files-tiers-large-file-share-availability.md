具有 100 TiB 容量限制的標準檔案共用，在全球所有 Azure 區域中都可供使用，除了：

- 本機多餘的儲存體：除了南非北部、南非西部、德國中西部和德國北部以外的所有區域。
- 區域冗余儲存體：支援區域冗余儲存體的所有區域（日本東部、北歐、南非北部除外）。
- 異地冗余/GeoZone 的多餘儲存體：不支援。
