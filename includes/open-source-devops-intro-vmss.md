[Azure 虛擬機器擴展集](../articles/virtual-machine-scale-sets/overview.md)是一項 Azure 功能，可讓您設定一組負載平衡的相同 VM。 擴展集沒有額外的成本，其是從虛擬機器建置而來。 您只需支付基礎計算資源的費用，例如 VM 執行個體、負載平衡器或受控磁碟儲存體。 透過擴展集將可提供管理和自動化層，以執行及調整您的應用程式。 您可改為手動建立和管理個別的 VM。 不過，使用擴展集有兩個主要優點。 它們內建在 Azure 中，自動調整您的虛擬機器以符合應用程式需求。
