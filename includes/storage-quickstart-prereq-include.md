若要存取 Azure 儲存體，您需要有 Azure 訂用帳戶。 如果您還沒有訂用帳戶，請先建立[免費帳戶](https://azure.microsoft.com/free/?WT.mc_id=A261C142F)，再開始操作。

對 Azure 儲存體的所有存取都是透過儲存體帳戶進行。 在本快速入門中，使用 [Azure 入口網站](https://portal.azure.com/)、Azure PowerShell 或 Azure CLI 建立儲存體帳戶。 如需建立帳戶的協助，請參閱[建立儲存體帳戶](../articles/storage/common/storage-quickstart-create-account.md)。
