若要開啟您在 Azure 中的函式應用程式的串流記錄檔：

1. 按下 F1 鍵開啟命令選擇區中，然後搜尋並執行命令**Azure Functions:啟動串流記錄**。

1. 在 Azure 中，選取您的函式應用程式，然後選取**是**啟用函式應用程式的應用程式記錄。

1. 您的函式，在 Azure 中，觸發程序，並注意記錄資料會顯示在 Visual Studio Code 中的 [輸出] 視窗。

1. 完成時，記得要執行此命令**Azure Functions:停止資料流記錄**停用函式應用程式的記錄。
