請檢查以判斷您是否擁有計費帳戶的存取權，Microsoft 客戶合約的合約類型。
 
1. 登入 [Azure 入口網站]( http://portal.azure.com)。
 
2. 針對 [成本管理 + 帳單]  進行搜尋。
 
   ![顯示 Azure 入口網站搜尋的螢幕擷取畫面](./media/billing-check-mca/billing-search-cost-management-billing.png)
 
3. 如果您只有一個帳單寄送的範圍內存取，請選取**屬性**從左上方。 您的計費帳戶的存取權的 Microsoft 客戶合約計費的帳戶類型是否**Microsoft 客戶合約**。
 
    ![在 [屬性] 頁面會顯示 microsoft 客戶合約的螢幕擷取畫面](./media/billing-check-mca/billing-mca-property.png)
 
4. 如果您需要存取多個計費的範圍，請檢查帳單的 [帳戶] 欄中的型別。 您的計費帳戶的存取權的 Microsoft 客戶合約的任何範圍的計費帳戶類型是否**Microsoft 客戶合約**。
 
    ![在 帳單的 帳戶 清單頁面上顯示 microsoft 客戶合約螢幕擷取畫面](./media/billing-check-mca/billing-mca-in-the-list.png)
