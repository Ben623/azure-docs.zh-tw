Azure Data Lake Storage Gen1 是容納巨量資料分析工作負載的企業級超大規模存放庫。 您可以使用 Data Lake 儲存體 Gen1 來擷取在進行運作和探究分析的單一位置擷取任何大小、 類型和擷取速度的資料。 您可以儲存在 Data Lake 儲存體 Gen1 帳戶中的資料量沒有限制。

| **Resource** | **預設限制** | **註解** |
| --- | --- | --- |
| Data Lake 儲存體 Gen1 帳戶，每個訂用帳戶，每個區域的最大數目 |10 | 若要要求增加此限制，請連絡支援。 |
| 每個檔案或資料夾的存取 Acl 數目上限 |32 | 這是一種硬性限制。 使用群組來管理存取權，以較少的項目。 |
| 每個檔案或資料夾的預設 Acl 數目上限 |32 | 這是一種硬性限制。 使用群組來管理存取權，以較少的項目。 |
