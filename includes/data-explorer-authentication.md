## <a name="authentication"></a>驗證

若要執行下列範例，您需要 Azure AD 應用程式和服務主體，才能存取資源。 請參閱[建立 Azure AD 應用程式](/azure/active-directory/develop/howto-create-service-principal-portal)，以建立免費的 Azure AD 應用程式，並在訂用帳戶層級新增角色指派。 您也需要 `directory (tenant) ID`、`application ID` 和 `client secret`。
