使用 Microsoft Azure 流量管理員將流量負載平衡到 Azure 網站時，接著可使用 Azure 指派的 **\*.trafficmanager.net** 網域名稱來存取該網站。 您也可以將自訂網域名稱，例如 www\.contoso.com，您的網站，以便提供更容易辨識的網域名稱，為您的使用者。
