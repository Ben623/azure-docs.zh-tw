# <a name="azure-maps-mobility-coverage"></a>Azure 地圖服務行動性涵蓋範圍

Azure 地圖服務[行動服務](https://aka.ms/AzureMapsMobilityService)允許公用傳輸、自行車共用和機車共用路由的功能，並提供更有階段的路線規劃選項。 對於公開傳輸，此服務會提供搜尋附近的工作站、停止和線路，並提供如排程和即時公共運輸抵達、停止和路線指南的清單，以及詳細和多重架構的逐步路線和服務警示。 行動服務也會傳回汽車分享車輛和共用自行車和 scooters 的即時和靜態可用性資訊。

Azure 地圖服務對於所有城市和國家/地區沒有相同層級的資訊和精確度。 呼叫傳輸資料的能力取決於城市，而資料可能不包含所有服務該城市的公開運輸選項和機構。

下表提供 Azure 地圖服務行動服務的涵蓋範圍資訊。


| 符號 | 意義 |
|--------|---------|
| *      |該國家/地區幾乎完全涵蓋。|
