# <a name="get-to-know-your-data-science-virtual-machine"></a>了解資料科學虛擬機器

資料科學虛擬機器 (DSVM) 包含熱門的資料科學、機器學習/AI 工具，以及資料平台的詳盡清單。 工具組態皆已經由 Microsoft 的資料科學家和開發人員，以及由更廣泛的資料科學社群進行完善的測試，以確保它們的穩定性和一般的可行性。

此 DSVM 也包含多個程式碼範例和逐步解說, 其形式為可在本機執行的 Jupyter 筆記本。此外, 還有這些範例的檔可協助理解。 

> [!NOTE]
> 您可以使用[Azure Notebooks](../../notebooks/azure-notebooks-overview.md)在雲端中建立、執行和共用 Jupyter 筆記本。 您可以[將資料科學 VM 附加至 Azure Notebooks](../../notebooks/configure-manage-azure-notebooks-projects.md#compute-tier)以擴充可用的計算能力。

包含在資料科學 VM 中的工具、平台、公用程式和範例，皆已整理至下列各節：

1. **[深度學習和 AI 架構](dsvm-deep-learning-ai-frameworks.md)**
2. **[機器學習與資料科學工具](dsvm-ml-data-science-tools.md)**
3. **[程式設計語言](dsvm-languages.md)**
4. **[開發工具和 IDE](dsvm-tools-development.md)**
5. **[資料擷取工具](dsvm-tools-ingestion.md)**
6. **[資料探索和虛擬化工具](dsvm-tools-explore-and-visualize.md)**
7. **[資料平台](dsvm-data-platforms.md)**
8. **[範例和逐步解說](dsvm-samples-and-walkthroughs.md)**
