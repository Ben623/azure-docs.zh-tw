# <a name="register-your-azure-digital-twins-app-with-azure-active-directory-legacy"></a>註冊 Azure 的數位對應項應用程式與 Azure Active Directory 舊版

本文章說明您可以使用的舊或傳統的方式註冊您的範例應用程式，以 Azure Active Directory (Azure AD)，使其可以存取您的數位對應項執行個體。 您可以嘗試這個方法，在案例中新的 Azure AD 應用程式註冊不適用於您的設定。
