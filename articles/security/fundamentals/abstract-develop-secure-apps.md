# <a name="develop-secure-cloud-applications-on-azure"></a>在 Azure 上開發安全的雲端應用程式
## <a name="abstract"></a>摘要
這份文件是一般指南，其中說明當您開發雲端應用程式時，您在軟體開發生命週期的每個階段所應考量的安全性問題和控制項。 在發行產品之前實作這些概念，可協助您建置更安全的軟體。 本文件中顯示的建議來自於我們在 Azure 安全性方面的經驗以及客戶的經驗。

這份文件的適用對象：所有層級的軟體設計人員、開發人員和測試人員，而這些人員會建置及部署安全的 Azure 解決方案。
