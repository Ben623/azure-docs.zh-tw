# <a name="threat-modeling-tool-update-release-71510231---1112018"></a>Threat Modeling Tool 更新版本 7.1.51023.1 - 11/1/2018

如同早先在 [GA 版本資訊](threat-modeling-tool-releases-71509112.md)中的說明，我們已發行 Microsoft Threat Modeling Tool 的更新 (7.1.51023.1)，要求預覽版本的使用者 (版本 < 7.1.50911.2 的預覽用戶端) 必須升級至支援的 GA 版本。 此版本不包含任何新功能或修正。

- 預覽版本的使用者將在用戶端開啟時自動下載升級。 如果您選擇不安裝新的更新，此工具的預覽版本就會關閉。
- 系統將會提示 GA 版工具的使用者選擇是否要升級。
- 2016 版工具的使用者將不受影響。

## <a name="feature-changes"></a>功能變更

- None

## <a name="system-requirements"></a>系統需求

- 支援的作業系統
  - Microsoft Windows 10
- 必要的 .NET 版本
  - .NET 3.5.2
- 其他需求
  - 必須有網際網路連線可接收工具和範本的更新。
