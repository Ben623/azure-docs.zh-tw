# <a name="create-a-consulting-offer"></a>建立諮詢服務供應專案

本文將說明如何建立及發佈諮詢服務供應項目。

## <a name="define-your-offer"></a>定義您的供應項目

定義您的套裝式諮詢服務供應項目。 專注於固定範圍、固定期間、估計的、固定的價格 (或免費) 和適用於單一客戶的售前導向供應項目。 選取受歡迎且有效為您帶來新業務的可重複套裝式業務開發。

## <a name="create-a-new-offer"></a>建立新的供應項目

若要建立新的供應項目，請依照下列步驟進行：
1. 在 Cloud Partner 入口網站的主功能表上，選取 [新增供應項目]。
1. 在 [新增供應項目] 功能表上，選取 [諮詢服務]。

    ![建立新的諮詢服務供應項目](media/cppselectnewconsultingoffer.png)
