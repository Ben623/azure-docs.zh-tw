# <a name="cloud-partner-portal-management"></a>Cloud Partner 入口網站管理

本章節描述的作業可用來建立及管理帳戶才能使用類型[Cloud Partner 入口網站](https://cloudpartner.azure.com)。  這些帳戶所需[Azure Marketplace](https://azuremarketplace.microsoft.com)並[AppSource](https://appsource.microsoft.com)。  


## <a name="background-information"></a>背景資訊

之前使用 Cloud Partner 入口網站，您應該熟悉下列文章：

- [Azure Marketplace 原則和詞彙](../../policies-terms.md)列出各種法律和技術需求和規則套用至的使用者和發行者[Azure Marketplace](https://azuremarketplace.microsoft.com)並[AppSource](https://appsource.microsoft.com)。
- [成為雲端 Marketplace 發行者](https://docs.microsoft.com/azure/marketplace/become-publisher)提供成為官方的 Microsoft 雲端發行者所需的所有步驟的概觀。


## <a name="account-creation-and-management"></a>帳戶建立和管理

本章節包含下列文章說明如何以管理 marketplace 發行者帳戶。

- [管理帳戶必要條件](./cpp-manage-prerequisites.md)說明如何成為註冊的雲端發行者，使用 CLoud Partner 入口網站的需求。
- [管理 Azure Marketplace 和 AppSource 發行者設定檔](./cpp-manage-publisher-profile.md)說明如何使用 Cloud Partner 入口網站來檢視和管理您的 Microsoft marketplace 發行者設定檔。 
- [Cloud Partner 入口網站上管理使用者](./cpp-manage-users.md)將使用者新增至您組織的雲端發佈帳戶，以及如何管理使用者存取權，以允許多人在使用相同的供應項目上運作。


## <a name="next-steps"></a>後續步驟

您必須符合[管理帳戶必要條件](./cpp-manage-prerequisites.md)再繼續進行。  之後，您也應該採取[雲端合作夥伴入口網站的導覽](../portal-tour/cpp-portal-tour.md)以熟悉入口網站的使用者介面。
