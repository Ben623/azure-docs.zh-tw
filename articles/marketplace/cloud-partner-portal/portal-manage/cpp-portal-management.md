# <a name="cloud-partner-portal-management"></a>Cloud Partner 入口網站管理

本節說明可用來建立及管理使用[Cloud Partner 入口網站](https://cloudpartner.azure.com)所需之帳戶的作業類型。  [Azure Marketplace](https://azuremarketplace.microsoft.com)和[AppSource](https://appsource.microsoft.com)都需要這些帳戶。  


## <a name="background-information"></a>背景資訊

在使用 Cloud Partner 入口網站之前，您應該先熟悉下列文章：

- [Azure Marketplace 原則和條款](../../policies-terms.md)列出各種法律和技術需求，以及將和發行者的使用者套用至[Azure Marketplace](https://azuremarketplace.microsoft.com)和[AppSource](https://appsource.microsoft.com)的規則。
- [成為雲端 Marketplace 發行者](https://docs.microsoft.com/azure/marketplace/become-publisher)概述成為官方 Microsoft 雲端發行者所需的所有步驟。


## <a name="account-creation-and-management"></a>帳戶建立和管理

本節包含下列文章，說明如何管理 marketplace 發行者帳戶。

- [管理帳戶的必要條件](./cpp-manage-prerequisites.md)說明如何成為已註冊的雲端發行者，這是使用 Cloud Partner 入口網站的需求。
- [管理 Azure Marketplace 和 AppSource 發行者設定檔](./cpp-manage-publisher-profile.md)說明如何使用 Cloud Partner 入口網站來查看及管理您的 Microsoft Marketplace 發行者設定檔。 
- [在 Cloud Partner 入口網站上管理使用者](./cpp-manage-users.md)將使用者新增至您組織的雲端發佈帳戶，以及如何管理其存取權，以允許多位人員使用相同的供應專案。


## <a name="next-steps"></a>後續步驟

在繼續之前，您必須符合[管理帳戶的必要條件](./cpp-manage-prerequisites.md)。  之後，您也應該進行[Cloud Partner 入口網站導覽](../portal-tour/cpp-portal-tour.md)，讓自己熟悉入口網站的使用者介面。
