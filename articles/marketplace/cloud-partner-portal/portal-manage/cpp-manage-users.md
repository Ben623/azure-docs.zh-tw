# <a name="managing-users-on-the-cloud-partner-portal"></a>管理 Cloud Partner 入口網站上的使用者

此[Cloud Partner 入口網站](https://cloudpartner.azure.com)可讓您新增具有虛擬機器供應專案之以角色為基礎之存取權的使用者。 這有助於管理多人使用供應項目時的存取與權限。

## <a name="add-users-and-assign-roles"></a>新增使用者和指派角色 

使用下列步驟來新增使用者，以及指派角色以進行供應專案發佈。

1. 按一下左側流覽窗格中的 [**使用者**] 索引標籤。

    ![使用者索引標籤](./media/userstab.png)


2. 按一下 [新增使用者]。

    ![選取 [新增使用者]](./media/adduser.png)


3. 輸入使用者的電子郵件地址，然後選取角色指派。  您可以將新使用者新增為「擁有者」或「參與者」。

    **電子郵件**︰新增即將發佈供應項目之組員的電子郵件地址。 支援 Microsoft 帳戶（Outlook、Hotmail 和 Live）和組織識別碼。

    - 新增「小組/群組電子郵件別名/安全性群組」，以防範離開組織的人員。
    - 請確定在 Cloud Partner 入口網站上提供的電子郵件識別碼會針對任何來自 Microsoft 的通訊進行監視。
    
    ![指派角色](./media/assignrole.png)

    **角色**︰請參閱下表，找出適當的使用者角色類型。

    ![角色層級](./media/roleaccesslevel.png)

    只有擁有者可以存取[賣方深入](../../cloud-partner-portal-orig/si-getting-started.md)解析中的 azure 支出和 azure 客戶索引標籤。


4. 選取您想要新增至新使用者的發行者名稱，然後按一下 [新增] 以完成使用者新增。

    > [!NOTE]
    > 將您的電子郵件新增為擁有者或參與者的 [發行者名稱]，將會顯示在 [**發行者]** 的下拉式清單中以供選取。


    ![指派角色 - 選取發行者](./media/assignselectpublisher.png)


## <a name="delete-an-existing-user"></a>刪除現有的使用者

若要移除現有的使用者，請在 [**使用者**] 索引標籤中搜尋其電子郵件，然後按右手邊的 [**刪除**] 按鈕。
