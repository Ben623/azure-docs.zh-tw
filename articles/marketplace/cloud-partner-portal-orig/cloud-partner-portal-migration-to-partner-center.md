# <a name="cloud-partner-portal-migration-to-partner-center"></a>Cloud Partner 入口網站遷移至合作夥伴中心

為了簡化您的帳戶並提供管理體驗，Microsoft 正在將 Cloud Partner 入口網站的供應專案遷移至合作夥伴中心。 如果您目前在一個商業夥伴的進入市場（OCP 轉型）目錄中管理共同銷售的宣傳，您很快就可以在合作夥伴中心更新您的共同銷售宣傳。

| **供應項目類型** | **從 Cloud Partner 入口網站遷移至合作夥伴中心的供應專案管理** | **能夠在合作夥伴中心更新共同銷售的宣傳** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| 軟體即服務（SaaS） |     現已推出       |   2020年2月正式推出      |
| Dynamics 365 for Customer Engagement & PowerApps      |     現已推出        |   現已推出      |
| Dynamics 365 for Operations     |     現已推出       |   現已推出     |
| Dynamics 365 Business Central      |     現已推出        |   2020年2月正式推出     |
| Azure 應用程式     |     正式運作12月2019       |   2020年2月正式推出     |
| Azure 虛擬機器      |     即將宣佈       |   即將宣佈    |
| Azure 容器     |     即將宣佈       |   即將宣佈     |
| 諮詢服務      |     即將宣佈       |   即將宣佈     |
| IoT Edge 模組      |     即將宣佈       |   即將宣佈     |
| Power BI 服務應用程式     |     即將宣佈       |   即將宣佈     |
|  |  |
