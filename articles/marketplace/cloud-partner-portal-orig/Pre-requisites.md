# <a name="prerequisites-for-application-curation"></a>應用程式鑑藏的先決條件

Microsoft 會要求特定的檢閱，來驗證是否符合下列需求：

- 合作夥伴的自訂程式碼符合 Microsoft 指導方針。\'
- 可成功部署 Microsoft Dynamics 週期服務 (LCS) 解決方案套件。
- 可以完成交易。

目前，合作夥伴必須透過執行測試部署，然後與 Microsoft 分享結果來表示已經滿足這些需求。 在 Microsoft 驗證之前，不會將任何程式碼部署到客戶環境。\' 合作夥伴必須完成下列鑑藏成品並測試：

- 程式碼分析報告 (CAR)
- 商務處理模組工具 (BPM)/測試指令碼
- 專案名稱和描述
- 商務資料庫備份
- 資料套件與處理資料套件 (PDP)
- 方法
- 二進位檔 (選擇性)
- 可部署的套件
- 模型 (程式碼和測試)
- 行銷內容

如需有關技術和程序需求的詳細資訊，請參閱[驗證應用程式 for Finance 與 Operations](https://docs.microsoft.com/dynamics365/operations/dev-itpro/lcs-solutions/app-validation-lcs-solutions)。  此外，如果您尚未這樣做，請檢閱[Office 365，Dynamics 365，PowerApps 及 Power BI 提供發佈指南](../appsource-offer-publishing-guide.md)。
