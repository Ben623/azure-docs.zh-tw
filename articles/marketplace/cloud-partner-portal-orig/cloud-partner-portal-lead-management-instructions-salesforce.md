# <a name="configure-lead-management-for-salesforce"></a>針對 Salesforce 設定潛在客戶管理

此文章說明如何設定 Salesforce 系統以處理銷售潛在客戶。

1. 登入 Salesforce。
2. 選取 [設定]  。
    ![Salesforce 設定](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce1.png)

3. 依序展開 [建置]  功能表、[自訂/潛在客戶/Web 到潛在客戶]  。
    ![Salesforce Web 到潛在客戶](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce2.png)

4. 在 [Web 到潛在客戶設定]  中，選取 [建立 Web 到潛在客戶表單]  。
    ![Salesforce 建立表單](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce3.png)

5. 在 [建立 Web 到潛在客戶表單]  上，選取 [產生]  ![Salesforce 產生](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce4.png)

6. 複製範例中的 OID 並儲存。 您要在 Cloud Partner 入口網站上的 [物件識別碼]  欄位中貼上 OID。
![Salesforce 取得 OID](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce5.png)
