# <a name="add-bank-and-tax-information"></a>新增銀行和稅務資訊  

所有使用交易清單類型的商業供應項目都必須有銀行和稅務資訊。  

## <a name="partner-center"></a>合作夥伴中心

現在會在合作夥伴中心內管理銀行和稅務資訊。 您可以在[管理合作夥伴中心帳戶](https://docs.microsoft.com/azure/marketplace/partner-center-portal/manage-account#financial-details)的「財務詳細資料」一節中找到設定這些設定的指示。
