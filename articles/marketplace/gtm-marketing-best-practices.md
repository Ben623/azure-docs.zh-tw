# <a name="marketing-best-practices"></a>行銷最佳做法

優化的 marketplace 清單和進入市場活動可以協助您加速客戶的取得成長，因此可以在雲端商務策略和引擎中扮演重要的角色。 準備好列出供應專案、提供客戶試用版，並與 Microsoft 客戶和合作夥伴的團體聯繫。 下載[AppSource & Azure Marketplace 行銷最佳做法指南](https://aka.ms/marketplacebestpractices)，以瞭解如何充分利用您的線上行銷工作。

**後續步驟**

若要深入了解 Marketplace GTM 服務，請移至 [Go-to-market Services](https://partner.microsoft.com/reach-customers/gtm)。

請登入 [Cloud Partner 入口網站](https://cloudpartner.azure.com)，以建立及設定您的供應項目。
