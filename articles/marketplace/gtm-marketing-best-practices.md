# <a name="marketing-best-practices"></a>行銷最佳做法

最佳化的 marketplace 清單和投入市場行銷活動可協助您加速您客戶的併購成長，並因此方面均扮演著重要的角色，在您的雲端商務策略和引擎。 準備就緒清單供應項目、 提供客戶試用版，並與 Microsoft 客戶和合作夥伴社群連線。 下載[AppSource 與 Azure Marketplace 行銷最佳做法指南](https://aka.ms/marketplacebestpractices)以了解如何充分運用您的線上行銷工作。

**後續步驟**

若要深入了解 Marketplace GTM 服務，請移至 [Go-to-market Services](https://partner.microsoft.com/reach-customers/gtm)。

請登入 [Cloud Partner 入口網站](https://cloudpartner.azure.com)，以建立及設定您的供應項目。
