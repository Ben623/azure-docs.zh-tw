# <a name="how-to-pay-your-bill-for-microsoft-azure"></a>如何在您的帳單支付 Microsoft Azure

這篇文章適用於與 Microsoft 客戶協議，透過 Azure 網站已註冊 Azure 的客戶。 [請檢查您的存取權的 Microsoft 客戶合約](#check-access-to-a-microsoft-customer-agreement)。

帳單設定檔的預設付款方法可以是一張信用卡，或檢查網路傳輸。 如果您有 Azure 信用額度時，這些會自動套用到您的發票每個計費週期。 若要成為合格的核取/電匯用多少付多少，請參閱[如何按發票付款](billing-how-to-pay-by-invoice.md)。

如果核取/電匯預設付款方法，您可以傳送您的款項匯到發票底部所列的地址。 或者，您可以用多少付多少線上使用信用卡，在 Azure 入口網站中。

如果預設付款方式的信用卡，我們會自動您信用卡收費每個計費期間之後會產生發票。 您可以也使用付款線上 Azure 入口網站中如果您想要使用不同的信用卡或自動收費失敗。

## <a name="pay-now-in-the-azure-portal"></a>在 Azure 入口網站中立即支付

您必須是計費的設定檔擁有者、 參與者或管理員發票支付發票仍在 Azure 入口網站。

1. 登入 [Azure 入口網站](https://portal.azure.com)。
1. 針對 [成本管理 + 帳單]  進行搜尋。
1. 在 [**概觀**刀鋒視窗中，按一下*摘要*] 索引標籤。

    ![顯示 [摘要] 索引標籤的螢幕擷取畫面](./media/billing-how-to-pay-bill/billing-account-overview.png)

1. 如果您有逾期未付，或是逾期未付帳款發票，您會看到總金額。
1. 按一下藍色**立即支付** 按鈕。

    ![螢幕擷取畫面顯示 [立即] 按鈕用多少付多少](./media/billing-how-to-pay-bill/paynow-button-overview.png)

1. 在新的刀鋒視窗中，按一下**選取付款方法**選擇現有的信用卡或新增帳戶。<!-- screenshot of select payment method link -->
1. 一旦選取付款方法時，按一下**立即支付**。

要支付發票刀鋒視窗中，將會更新發票的狀態。
