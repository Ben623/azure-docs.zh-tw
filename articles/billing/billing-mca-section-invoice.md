# <a name="create-sections-on-your-invoice-to-organize-your-costs"></a>若要組織您的成本您發票上建立區段

建立依部門時，開發環境中，組織您的成本您發票上的區段，或根據您組織的需求。 然後讓其他人建立區段的 Azure 訂用帳戶的權限。 任何使用費用或訂用帳戶的購買項目會反映 [發票] 區段。 您在發票上，在 Azure 入口網站中，檢視區段的總費用或 Azure 的成本分析中檢閱它們。 如需詳細資訊，請參閱 <<c0> [ 檢視發票區段的最前面交易](billing-mca-understand-your-bill.md#view-transactions-by-invoice-sections)。

這篇文章適用於計費的帳戶 Microsoft 客戶合約。 [檢查您是否擁有存取權的 Microsoft 客戶合約](#check-access-to-a-microsoft-customer-agreement)。

## <a name="create-an-invoice-section-in-the-azure-portal"></a>在 Azure 入口網站中建立的發票區段

若要建立的 「 發票 」 區段，您必須要**計費的設定檔擁有者**或是**計費的設定檔參與者**。 如需詳細資訊，請參閱 <<c0> [ 管理計費的設定檔的發票區段](billing-understand-mca-roles.md#manage-invoice-sections-for-billing-profile)。

1. 登入 [Azure 入口網站](https://portal.azure.com)。

2. 針對 [成本管理 + 帳單]  進行搜尋。

   ![顯示 Azure 入口網站搜尋的螢幕擷取畫面](./media/billing-mca-section-invoice/billing-search-cost-management-billing.png)

3. 選取 **發票區段**左側窗格中。 根據您的存取權，您可能需要選取計費的設定檔或帳單帳戶，然後選取**發票區段**。

   ![顯示 Azure 入口網站搜尋的螢幕擷取畫面](./media/billing-mca-section-invoice/billing-mca-list-invoice-sections.png)

4. 從頁面頂端選取 [新增]  。

5. 輸入 [invoice] 區段的名稱。

   ![顯示 [發票] 區段建立頁面的螢幕擷取畫面](./media/billing-mca-section-invoice/mca-create-invoice-section.png)

6. 選取計費的設定檔。 任何使用量費用或購買發票區段會反映在帳單設定檔的發票上。

7. 選取 [建立]  。
