# <a name="troubleshoot-a-declined-card-at-azure-sign-up"></a>在 Azure 註冊時針對已拒絕的卡片進行疑難排解

本文可協助您解決在 Azure 入口網站或 Azure 帳戶中心的 Azure 註冊時, 信用卡遭到拒絕的問題。 開始針對問題進行疑難排解之前, 請檢查下列各點:

- 請確定您為 Azure 帳戶設定檔提供的資訊 (例如連絡人電子郵件、位址和電話號碼) 正確無誤。
- 請確認信用卡資訊正確。
- 確定您沒有具有相同資訊的 Microsoft 帳戶。
- 不接受轉帳卡

## <a name="issues"></a>問題

以下列出可能會在 Azure 註冊時拒絕信用卡的常見問題。

### <a name="the-credit-card-provider-is-not-accepted-for-your-country"></a>您的國家/地區不接受信用卡提供者

當您選擇卡片時, Azure 會顯示在您選取的國家/地區中有效的卡片選項。 請洽詢您的銀行或持卡簽發者, 以確認您的信用卡已啟用國際交易。 如需有關支援的國家/地區和貨幣的詳細資訊, 請參閱[Azure 購買常見問題](https://azure.microsoft.com/pricing/faq/)。

>[!Note]
>美國 Express 信用卡目前不支援印度的付款條件。 我們沒有時間範圍, 因為它可能是接受的付款形式。

### <a name="youre-using-a-virtual-or-prepaid-card"></a>您使用的是虛擬或預付卡 

虛擬卡、預付信用卡或金融卡不被視為 Azure 訂用帳戶的付款方式。

### <a name="your-credit-information-is-inaccurate-or-incomplete"></a>您的信用額度資訊不正確或不完整 

您輸入的名稱、位址和 CVV 碼必須完全符合卡片上列印的內容。

### <a name="the-card-is-inactive-or-blocked"></a>卡片處於非使用中或已封鎖 

請洽詢您的銀行, 以確定您的卡片為使用中狀態。

您可能遇到其他註冊問題 

如需有關如何針對 Azure 註冊問題進行疑難排解的詳細資訊, 請參閱下列知識庫文章: 

[您無法在 Azure 入口網站或 Azure 帳戶中心註冊 Azure](billing-troubleshoot-azure-sign-up.md)

### <a name="you-represent-a-business-that-doesnt-want-to-pay-by-card"></a>您代表不想要以卡片支付的公司 

如果您代表企業, 您可以使用發票付款方法 (例如支票、夜間檢查或傳輸線路) 來支付您的 Azure 訂用帳戶。 將帳戶設定為按發票付款之後, 除非您擁有 Microsoft 客戶合約並透過 Azure 網站註冊 Azure, 否則無法變更為其他付款選項。

如需如何按發票付款的詳細資訊, 請參閱[提交依發票支付 Azure 訂用帳戶的要求](billing-how-to-pay-by-invoice.md)。

### <a name="your-credit-card-information-is-outdated"></a>您的信用卡資訊已過期 

如需如何管理您的卡片資訊 (包括變更或移除卡片) 的相關資訊, 請參閱[新增、更新或移除 Azure 的信用額度](billing-how-to-change-credit-card.md)。
