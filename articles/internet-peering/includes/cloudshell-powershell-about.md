若要執行 Cmdlet，您可以使用 Azure Cloud Shell (免費的互動式殼層)。 它具有預先安裝和設定的共用 Azure 工具，可與您的帳戶搭配使用。 只要按一下 [複製] 即可複製程式碼，將它貼到 Cloud Shell 中，然後按 enter 鍵加以執行。 以下有幾種啟動 Cloud Shell 的方式：


|  |   |
|-----------------------------------------------|---|
| 在您的瀏覽器中開啟 Cloud Shell。 | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| 按一下入口網站右上方功能表上的 [ **Cloud Shell** ] 按鈕。 | [![入口網站中的 Cloud Shell](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


如果您不想要使用 Azure CloudShell，您可以改為在本機安裝 PowerShell。 如果您選擇在本機安裝和使用 PowerShell，請務必安裝最新版的 Resource Manager PowerShell Cmdlet。 PowerShell Cmdlet 會經常更新，您通常需要更新您的 PowerShell Cmdlet，以取得最新的功能，失敗，您可能會遇到問題。

若要尋找您在本機執行的 PowerShell 版本，請使用 ' ListAvailable Az ' Cmdlet。 若要更新，請參閱[安裝 Azure PowerShell 模組](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps)。 如需詳細資訊，請參閱 [如何安裝及設定 Azure PowerShell](https://docs.microsoft.com/powershell/azure/azurerm/overview)。

如果您在 macOS 上使用 PowerShell，請遵循在[macOS 上安裝 Powershell Core](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6)中的步驟。
