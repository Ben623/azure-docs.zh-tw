## <a name="languages---community-supported"></a>程式語言 - 社群支援

有幾個社群支援 Application Insights Sdk，而其中有許多原本由 Microsoft 所撰寫。 社群支援的 Sdk 不會正式維護由 Microsoft 和我們目前無法提供任何未正式支援的清單的 SDK 支援。 這些 Sdk 會被視為實驗階段，並不建議用於生產環境。
