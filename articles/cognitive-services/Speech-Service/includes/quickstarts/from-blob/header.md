在本快速入門中，您將使用 REST API 於批次程序中辨識檔案內的語音。 批次程序不需要任何使用者互動就會執行語音轉譯。 其會提供您簡單的程式設計模型，讓您不必管理並行、自訂語音模型或其他詳細資料。 其需要進階的控制選項，才能有效率地使用 Azure 語音服務資源。

[批次轉譯概觀](../../../batch-transcription.md)會說明此功能在使用上的詳細資料。 詳細的 API 可於 `Custom Speech transcriptins` 標題底下的 [Swagger 文件](https://westus.cris.ai/swagger/ui/index#/Custom%20Speech%20transcriptions%3A)中取得。 

下列快速入門會逐步引導您完成使用範例。
