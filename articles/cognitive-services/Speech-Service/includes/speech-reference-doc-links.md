## <a name="reference-docs"></a>參考文件

語音服務提供兩個 Sdk。 第一個 SDK 是主要的[語音 sdk](../speech-sdk.md) ，並提供與語音服務互動所需的大部分功能。 第二個 SDK 是裝置特有的，其已適當命名為[語音裝置 SDK](../speech-devices-sdk.md)。 這兩種 Sdk 都提供許多語言。

使用下列清單來尋找適當的語音 SDK 參考檔：

> 語音服務 SDK 會主動維護和更新。 若要追蹤變更，更新和新增功能會參考[語音 SDK 版本](../releasenotes.md)資訊。

### <a name="speech-devices-sdk-reference-docs"></a>語音裝置 SDK 參考檔

[語音裝置 SDK](../speech-devices-sdk.md)是語音 SDK 的超集合，具有適用于特定裝置的擴充功能。 若要下載語音裝置 SDK，您必須先[選擇開發工具組](../get-speech-devices-sdk.md#choose-a-development-kit)。
