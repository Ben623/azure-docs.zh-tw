在 LUIS 入口網站的 [Azure 資源] 頁面 [管理] 區段中，提供金鑰和端點的存取權。 
 
1. 登入 LUIS 入口網站。 
1. 從應用程式清單中選取應用程式。
1. 選取 [管理]  ，然後選取 [Azure 資源]  。 
1. 複製入門資源的金鑰值和端點值。

    **入門**金鑰允許對撰寫 API 提出免費要求，而且每個月可對所有 LUIS 應用程式的預測端點 API 進行多達 1000 次查詢。
