## <a name="select-subscription-and-authoring-resource"></a>選取訂用帳戶和製作資源

在 LUIS 入口網站中，若要查看應用程式清單或建立應用程式，請選取訂用帳戶和製作資源。 應用程式會透過 LUIS 製作資源來存取。
