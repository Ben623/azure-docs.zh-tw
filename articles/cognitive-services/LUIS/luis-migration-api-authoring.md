# <a name="authoring-api-changes"></a>撰寫 API 變更 

撰寫 Api 會針對 V3 進行更新，以允許[使用機器學習的實體進行模型分解](luis-concept-data-extraction.md)。 

您可以在 V2 或預覽 V3 API 中撰寫。 相同的應用程式也可以選擇使用 V2 或 V3 預測端點回應。 您不需要有相同版本的 API 來撰寫和預測端點回應。

LUIS 入口網站會以獨佔方式使用 V3 撰寫 Api。

## <a name="next-steps"></a>後續步驟

審查 V3 API[的預測端點變更](luis-migration-api-v3.md)。
