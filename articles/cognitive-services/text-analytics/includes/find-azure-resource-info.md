> [!TIP]
> 若要在 Azure 入口網站上尋找您的金鑰和端點：
> 1. 在 https://portal.azure.com/ 上瀏覽至您的 Azure 資源。
> 2. 按一下位於 [資源管理]  下方的 [快速入門]  。
>
> 請務必從您的端點中移除 `/text/analytics/<version>` 尾碼 (如果有的話)。 用戶端程式庫不需使用此尾碼。
