# <a name="quickstart-use-the-text-analytics-client-library-v2"></a>快速入門：使用文字分析用戶端程式庫 v2

開始使用文字分析用戶端程式庫 v2。 請遵循下列步驟來安裝套件，並試用基本工作的程式碼範例。

使用文字分析用戶端程式庫 v2 來執行：

* 情感分析
* 語言偵測
* 實體辨識
* 關鍵片語擷取
