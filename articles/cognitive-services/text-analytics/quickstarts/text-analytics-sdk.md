# <a name="quickstart-use-the-text-analytics-client-library-v3"></a>快速入門：使用文字分析用戶端程式庫 v3

開始使用文字分析用戶端程式庫。 請遵循下列步驟來安裝套件，並試用基本工作的程式碼範例。

使用文字分析用戶端程式庫版本 `3.0-preview` 來執行：

* 情感分析 (公開預覽)
* 語言偵測
* 具名實體辨識 (公開預覽)
* 關鍵片語擷取
