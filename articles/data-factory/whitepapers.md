# <a name="azure-data-factory-whitepapers"></a>Azure Data Factory 白皮書 

白皮書可讓您以更深入的層面探索 Azure Data Factory。 本文提供 Azure Data Factory 的可用白皮書清單。

| **白皮書** | **描述** |
| --- | --- |
|[Azure Data Factory-雲端中的資料整合](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | 本檔說明 Azure Data Factory 如何讓您建立現代化的資料倉儲、啟用先進的分析以驅動智慧型 SaaS 應用程式, 並將您的 SQL Server 整合服務套件提升至 Azure。|
|[使用 Azure Data Factory 從內部部署關聯式資料倉儲將資料移轉至 Azure](https://azure.microsoft.com/mediahandler/files/resourcefiles/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/Data_migration_from_on-prem_RDW_to_ADLS_using_ADF.pdf) | 本檔說明使用 Azure Data Factory 將數十 TB 資料從現有內部部署關聯式資料倉儲 (例如, Netezza、Oracle、Teradata、SQL server) 遷移至 Azure (例如 Blob 儲存體或 Azure Data Lake Storage) 的複雜性。 這些挑戰和最佳作法會以彈性、效能、擴充性、管理和安全性說明, 以透過 Azure Data Factory 來進行 Azure 的大規模資料內嵌旅程。 |
|[使用 Azure Data Factory 的 SAP 資料整合](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | 本文摘要說明 Azure Data Factory 目前對 SAP 資料整合的支援, 包括目標案例、適用于不同需求的 SAP 連接器選項和比較, 以及 Data Factory 中每個 SAP 連接器的簡介。 | 
|[Azure Data Factory:雲端中的 SSIS](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| 這份檔會說明為什麼您會想要將現有的 SSIS 工作負載遷移至 Azure Data Factory, 並解決常見的考慮和顧慮。 接著, 我們將逐步引導您建立 Azure SSIS IR 的技術詳細資料, 並示範如何透過 Azure Data Factory 使用您可能熟悉的工具 (例如 SQL Server Management Studio (SSMS)), 來上傳、執行和監視您的套件。 |
|[Azure data Factory-傳遞參數](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| 本文會逐步引導您瞭解如何在管線和活動之間, 以及活動之間傳遞參數。 |
|[Azure data Factory-DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| 本白皮書涵蓋一些關於持續整合和部署 Azure Data Factory 的最佳作法。 |
