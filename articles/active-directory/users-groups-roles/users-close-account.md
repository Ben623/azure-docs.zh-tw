# <a name="close-your-work-or-school-account-in-an-unmanaged-directory"></a>關閉您的公司或學校帳戶，在非受控目錄

如果您已在使用者於未受管理的 Azure Active Directory (Azure AD) 組織，您不再需要使用來自該組織的應用程式或維護的任何關聯的是，您可以隨時關閉您的帳戶。 非受控的目錄沒有全域管理員。 非受控目錄中的使用者可以關閉他們的帳戶，其本身，而不需要連絡系統管理員。

非受控目錄中的使用者通常會建立在自助期間註冊。 範例可能是知識工作者在組織中註冊免費的服務。 如需有關自助式註冊的詳細資訊，請參閱[什麼是自助式 Azure Active Directory 註冊？](directory-self-service-signup.md)。

[!INCLUDE [GDPR-related guidance](../../../includes/gdpr-intro-sentence.md)]

## <a name="before-you-begin"></a>開始之前

您可以關閉您的帳戶之前，您應該確認下列項目：

* 請確定您是使用者的非受控 Azure AD 目錄。 如果您屬於受控目錄，則無法關閉您的帳戶。 如果您屬於受管理的目錄，並想要關閉您的帳戶，您必須連絡您的系統管理員。 如需有關如何判斷是否屬於非受控目錄的資訊，請參閱[從非受控租用戶刪除使用者](https://docs.microsoft.com/flow/gdpr-dsr-delete#delete-the-user-from-unmanaged-tenant)。

* 儲存您想要保留的任何資料。 如需如何提交匯出要求資訊，請參閱[存取和匯出系統產生記錄檔未受管理的租用戶](https://docs.microsoft.com/power-platform/admin/powerapps-gdpr-dsr-guide-systemlogs#accessing-and-exporting-system-generated-logs-for-unmanaged-tenants)。

> [!WARNING]
> 關閉您的帳戶是無法復原。 當您關閉您的帳戶時，將會移除所有個人資料。 您將不再有權限帳戶和您的帳戶相關聯的資料。

## <a name="close-your-account"></a>關閉您的帳戶

若要關閉的未受管理的工作或學校帳戶，請遵循下列步驟：

1. 登入[關閉您的帳戶](https://go.microsoft.com/fwlink/?linkid=873123)，使用您想要關閉的帳戶。

1. 在 **我的資料要求**，選取**關閉帳戶**。

    ![我的資料要求-關閉帳戶](./media/users-close-account/close-account.png)

1. 檢閱確認訊息，然後選取**是**。

    ![我的資料要求數-關閉確認](./media/users-close-account/confirm-close.png)
