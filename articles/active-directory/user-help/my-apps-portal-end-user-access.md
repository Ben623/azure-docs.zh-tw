# <a name="access-and-use-apps-on-the-my-apps-portal"></a>存取和使用 My Apps 入口網站上的應用程式
您可以透過 [我的應用程式]  Web 入口網站使用工作或學校帳戶，來檢視和啟動貴組織的許多雲端式應用程式、更新您的設定檔和帳戶資訊、查看您的**群組**資訊，以及執行您應用程式和群組的**存取權檢閱**。 如果您沒有 [我的應用程式]  入口網站的存取權，則必須連絡技術服務人員以取得權限。

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

>[!Important]
>本內容適用於使用者。 如果您是系統管理員，可以在[應用程式管理文件](https://docs.microsoft.com/azure/active-directory/manage-apps)中找到更多關於如何設定和管理雲端式應用程式的資訊。

## <a name="supported-browsers"></a>支援的瀏覽器
您可以前往**My Apps**入口網站從任何下列的網頁瀏覽器：

- Google Chrome

- Mozilla Firefox 26.0 或更新版本的版本

- Microsoft Edge

- Internet Explorer 11 （支援有限） 版

## <a name="download-and-install-the-my-apps-secure-sign-in-extension"></a>下載並安裝 My Apps 安全登入延伸模組
下載並安裝 My Apps 安全登入延伸模組，如果系統提示您。 此延伸模組可協助您啟動任何組織的雲端應用程式，您必須使用單一登入程序。 如果您的組織具有已設定以進行單一登入，延伸模組會自動安裝，您可以略過本節。

此延伸模組可協助您： 

- 從登入直接應用程式的登入頁面。

- 開始使用的任何應用程式**快速搜尋**功能。

- 請參閱您使用中的最後一個應用程式**最近使用過**一節。

- 使用公司內部 Url 時使用的遠端[應用程式 Proxy](https://docs.microsoft.com/azure/active-directory/active-directory-application-proxy-get-started)。

### <a name="to-download-and-install-the-extension"></a>若要下載並安裝擴充功能
下載並安裝擴充功能，根據您使用的瀏覽器：

- **Google Chrome。** 從 Chrome Web Store 中，移至[My Apps 安全登入延伸模組](https://chrome.google.com/webstore/detail/my-apps-secure-sign-in-ex/ggjhpefgjjfobnfoldnjipclpcfbgbhl)功能，然後按**加入 Chrome**。

- **Mozilla Firefox**從**Firefox 附加元件**頁面上，移至[My Apps 安全登入延伸模組](https://addons.mozilla.org/firefox/addon/access-panel-extension/)功能，然後按**加入 Firefox**。

- **Microsoft Edge**從 Microsoft Store，請移至[My Apps 安全登入延伸模組](https://www.microsoft.com/p/my-apps-secure-sign-in-extension/9pc9sckkzk84?rtc=1&activetab=pivot%3Aoverviewtab)功能，然後按**取得**。

圖示會新增至右側您**地址**列上，讓您登入和自訂延伸模組。

#### <a name="to-change-your-my-apps-portal-using-the-extension"></a>若要變更您的 My Apps 入口網站，使用擴充功能
您可以選擇檢視中有多少應用程式**最近使用過**區段，並決定是否要允許您的組織內部的 Url 重新導向。

1. 選取新**My Apps 安全登入延伸模組**圖示![延伸模組圖示](media/my-apps-portal/my-apps-portal-extension-icon.png)右邊您**位址**列，，然後選取**登入以取得啟動的**.

2. 以滑鼠右鍵按一下**設定**圖示![設定圖示](media/my-apps-portal/my-apps-portal-extension-settings-icon.png)，然後選取**設定**。

3. 在 **設定**，選取您想要在入口網站中，請參閱最近的應用程式的數目，是否允許您的組織的內部 Url 重新導向，您可以使用這些遠端。

    ![顯示可用的自訂延伸模組的 [設定] 頁面](media/my-apps-portal/my-apps-portal-extension-settings-page.png)

## <a name="access-and-use-the-my-apps-portal-by-device"></a>存取和使用 My Apps 入口網站中的裝置
您可以存取，並使用您的電腦，從受 Intune 管理的瀏覽器中，或從 iOS 或 Android 行動裝置上的 My Apps 入口網站。

![在 My Apps 入口網站中的 [應用程式] 頁面](media/my-apps-portal/my-apps-portal-apps-page.png)

### <a name="access-and-use-the-my-apps-portal-on-your-computer"></a>存取和使用 My Apps 入口網站，在您的電腦上
如果您有權限以存取並使用您組織的雲端式應用程式時，您能透過他們**My Apps**入口網站。

1.  登入您的公司或學校帳戶。

2.  開啟網頁瀏覽器並移至 https://myapps.microsoft.com ，或使用貴組織所提供的連結。 比方說，您可能會被導向至自訂頁面為您的組織，例如 https://myapps.microsoft.com/contoso.com 。

    **應用程式**頁面隨即出現，顯示的所有雲端應用程式，您的組織所擁有且可供您使用。

3. 從**應用程式**頁面上，選取您想要開始使用的應用程式。
    
    新的頁面會開啟應用程式中，您可以登入 （如有必要） 或開始使用應用程式。

### <a name="access-and-use-the-my-apps-portal-on-an-intune-managed-browser"></a>存取和使用 My Apps 入口網站，在受 Intune 管理的瀏覽器上
檢視及使用貴組織的 Intune Managed Browser 應用程式在您的 iOS 和 Android 裝置上。

1. 您的行動裝置上下載並安裝 Intune Managed Browser 應用程式，從[Apple App Store](https://itunes.apple.com/us/app/microsoft-intune-managed-browser/id943264951?mt=8)並[Google Play 商店](https://play.google.com/store/apps/details?id=com.microsoft.intune.mam.managedbrowser)。

2. 開啟 Intune Managed Browser 應用程式，請移至 https://myapps.microsoft.com ，或使用貴組織所提供的連結。 比方說，您可能會被導向至自訂頁面為您的組織，例如 https://myapps.microsoft.com/contoso.com 。

    **應用程式**頁面隨即出現，顯示的所有雲端應用程式，您的組織所擁有且可供您使用。

3. 從**應用程式**頁面上，選取您想要開始使用的應用程式。
    
    新的頁面會開啟應用程式中，您可以登入 （如有必要） 或開始使用應用程式。

## <a name="add-a-new-app-to-the-my-apps-portal"></a>My Apps 入口網站中加入新的應用程式
如果您的系統管理員指定的權限，您可以加入新的應用程式**應用程式**頁面。

1. 從**應用程式**頁面上，選取**新增應用程式**。

    **將應用程式新增**頁面隨即出現。

    ![在 [My Apps 入口網站中新增應用程式] 頁面中，](media/my-apps-portal/my-apps-portal-add-apps-page.png)

2. 選取您想要新增從提供的清單，然後選取 應用的程式**新增**。

3. 應用程式新增至清單上**應用程式**頁面。

    某些應用程式可能需要系統管理員核准，才能新增。 在此情況下，應用程式不會新增至**應用程式**頁面上，直到系統管理員核准它。
