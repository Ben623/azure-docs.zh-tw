# <a name="my-account-portal-overview"></a>[我的帳戶] 入口網站概觀

[我的帳戶]  (預覽) 入口網站可藉由設定和管理安全性資訊、管理已連線的組織和裝置，以及檢視組織使用資料的方式，協助您管理公司或學校帳戶。

>[!Important]
>本內容適用於使用者。 I如果您是系統管理員，可以在 [Azure Active Directory 文件](https://docs.microsoft.com/azure/active-directory)中找到更多關於如何設定和管理 Azure Active Directory (Azure AD) 環境的資訊。

![[我的帳戶] 入口網站，[概觀] 頁面](media/my-account-portal/my-account-portal-overview.png)

## <a name="go-to-the-my-account-portal"></a>前往 [我的帳戶] 入口網站

您可以從下列任何瀏覽器目前的版本前往 [我的帳戶]  入口網站 (https://myprofile.microsoft.com) )：

- Chrome
- Microsoft Edge
- Safari
- Firefox
- Internet Explorer 11
