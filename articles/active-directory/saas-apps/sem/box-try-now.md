# <a name="box-and-azure-active-directory"></a>Box 和 Azure Active Directory

加速採用整個組織的箱, 並透過我們預先建立的整合, 提升使用者的採用。 藉由將 Box 連接至 Azure Active Directory (Azure AD), 讓使用者能夠快速進行共同作業。

![顯示 Box 應用程式標誌](./media/box.png)

> [!div class="nextstepaction"]
> [使用 Azure AD > 試用 Box](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>Box 和 Azure AD 可以做什麼？

- **簡單的單一登入**-排除多個使用者名稱和密碼, 並允許使用者使用他們的 Azure AD 帳戶登入 Box, 以簡化存取。
- **自動化使用者**布建-當使用者加入、移動或離開您的組織時, 自動建立、維護和移除使用者帳戶到 Box。
- 藉由套用條件式存取原則、透過存取權審查來管理存取權, 以及使用登入記錄來監視存取權, 來**保護、管理及監視對**Box 的存取安全存取。

## <a name="get-started"></a>開始使用

透過我們的簡單設定指南, 為 Box 設定[使用者](https://aka.ms/BoxSSODoc)布[建和單一登入](https://aka.ms/BoxProvisioningDoc)。
