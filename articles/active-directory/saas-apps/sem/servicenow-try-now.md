# <a name="servicenow-and-azure-active-directory"></a>ServiceNow 和 Azure Active Directory

加速採用整個組織的 ServiceNow, 並透過我們預先建立的整合, 提升使用者的採用。 將 ServiceNow 連接到 Azure Active Directory (Azure AD), 以簡化 IT 服務管理。

![顯示 ServiceNow 應用程式標誌](./media/servicenow.png)

> [!div class="nextstepaction"]
> [使用 Azure AD > 嘗試 ServiceNow](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>您可以使用 ServiceNow 和 Azure AD 做什麼？

- **簡單的單一登入**-排除多個使用者名稱和密碼, 並藉由允許使用者使用他們的 Azure AD 帳戶登入 ServiceNow, 以簡化存取。
- **自動化使用者**布建-當使用者加入、移動或離開您的組織時, 將使用者帳戶的建立、維護和移除作業自動化至 ServiceNow。
- 藉由套用條件式存取原則、透過存取權審查來管理存取權, 以及使用登入記錄來監視存取, 來**保護、管理及監視對**ServiceNow 的存取安全存取。

## <a name="get-started"></a>開始使用

透過我們的簡單設定指南, 為 ServiceNow 設定[使用者](https://aka.ms/ServiceNowProvisioning)布[建和單一登入](https://aka.ms/ServiceNowSSO)。
