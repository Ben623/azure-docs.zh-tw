# <a name="workday-and-azure-active-directory"></a>Workday 和 Azure Active Directory

使用我們預先建立的整合, 將 Workday 與 Azure Active Directory (Azure AD) 連接, 並在員工加入、移動或離開時, 自動布建或取消布建使用者帳戶。

![顯示 Workday 應用程式標誌](./media/workday.png)

> [!div class="nextstepaction"]
> [使用 Azure AD > 試用 Workday](https://aka.ms/WorkdayTryItNow)

## <a name="what-can-you-do-with-workday-and-azure-ad"></a>Workday 和 Azure AD 可以做什麼？

- **自動化使用者**布建-根據 Workday 中的群組、角色或其他屬性, 自動將應用程式指派給員工。
- **同步處理員工設定檔**-確保員工資料在 Azure AD 和 Workday 之間保持在最新狀態。
- **簡化上線和脫離**-自動布建新進員工的使用者帳戶, 讓他們可以在一天內有效率地工作, 並在員工離職時自動取消布建使用者帳戶, 以降低安全性風險。
- **輕鬆單一登入**排除使用者名稱和密碼, 並允許使用者使用其裝載在 Azure AD 中的組織帳戶登入。

## <a name="get-started"></a>開始使用

透過我們的簡單設定指南, 為 Workday 設定[使用者](https://aka.ms/WorkdayProvisioning)布[建和單一登入](https://aka.ms/WorkdaySSODoc)。
