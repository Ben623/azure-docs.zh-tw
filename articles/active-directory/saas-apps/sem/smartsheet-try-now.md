# <a name="smartsheet-and-azure-active-directory"></a>Smartsheet 和 Azure Active Directory

加速採用整個組織的 Smartsheet, 並透過我們預先建立的整合, 提升使用者的採用。 藉由將 Smartsheet 連接到 Azure Active Directory (Azure AD), 讓使用者能夠快速共同作業。

![顯示 Smartsheet 應用程式標誌](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [使用 Azure AD > 嘗試 Smartsheet](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>您可以使用 Smartsheet 和 Azure AD 做什麼？

- **簡單的單一登入**-排除多個使用者名稱和密碼, 並藉由允許使用者使用他們的 Azure AD 帳戶登入 Smartsheet, 來簡化存取。
- **自動化使用者**布建-當使用者加入、移動或離開您的組織時, 將使用者帳戶的建立、維護和移除作業自動化至 Smartsheet。
- 藉由套用條件式存取原則、透過存取權審查來管理存取權, 以及使用登入記錄來監視存取權, 來**保護、管理及監視存取**安全的 Smartsheet 存取。

## <a name="get-started"></a>開始使用

設定 Smartsheet 的[使用者](https://aka.ms/SmartsheetProvisioningDoc)布建和[單一登入](https://aka.ms/SmartsheetSSODoc), 並提供我們的簡單設定指南。
