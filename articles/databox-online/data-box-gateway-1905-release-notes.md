# <a name="azure-data-box-edge-and-azure-data-box-gateway-1905-release-notes"></a>Azure 資料方塊邊緣和 Azure 資料方塊閘道 1905年版本資訊

## <a name="overview"></a>概觀

下列版本資訊識別的重大未決問題和 1905年版 Azure 資料方塊邊緣和 Azure 中部署資料閘道已解決的問題。

版本資訊會持續更新，並在發現需要提出因應措施的重大問題時有所增補。 部署您的資料方塊邊緣/資料方塊閘道之前，請仔細檢閱版本資訊中所包含的資訊。

這個版本會對應到軟體版本：

- **資料方塊閘道 1905 (1.6.887.626)**
- **資料方塊邊緣 1905 (1.6.887.626)**

> [!NOTE]
> 更新 1905年只用於資料方塊的邊緣裝置執行 GA 軟體版本。

## <a name="whats-new"></a>新功能

- **欄位可程式化 Gate Array (FPGA) 記錄的改善**-在此版本中，我們已記錄及警示的增強功能的相關 FPGA。 如果您使用 FPGA Edge 計算功能，這會是資料方塊邊緣的必要的更新。 如需詳細資訊，請參閱如何[使用您資料的方塊邊緣上的 Edge 計算來轉換資料](data-box-edge-deploy-configure-compute-advanced.md)。

## <a name="known-issues-in-ga-release"></a>GA 版本的已知的問題

沒有新的問題是此版本中所述的版本。 所有版本中，記都下問題即已存在從之前的版本。 若要查看已知問題清單，請前往[GA 版本中的已知問題](data-box-gateway-release-notes.md#known-issues-in-ga-release)。
