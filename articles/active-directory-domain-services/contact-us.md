# <a name="azure-ad-domain-services---contact-us"></a>Azure AD 網域服務 - 與我們連絡

## <a name="feedback"></a>意見反應
如果您有意見反應，我們的服務或非技術性問題，您可以使用 Azure AD 網域服務產品小組共用您的要求，以電子郵件傳送給我們： 項目：[Azure AD Domain Services 意見反應](mailto:aaddsfb@microsoft.com)。 檢閱電子郵件至此地址。 產品群組的成員**可能**回應要求進一步的資訊。 將不會回應此電子郵件地址的技術支援要求。

## <a name="technical-assistance"></a>技術協助
如果對您的受控網域有問題，請檢查以了解 [疑難排解指南](troubleshoot.md) 中所述的步驟是否可解決問題。 如果您仍然遇到問題，請登入 Azure 入口網站。 開啟**Azure AD Domain Services**首頁，然後選取**新增支援要求**從底下的 [動作] 窗格**支援與疑難排解**。

![新增支援要求](./media/contact-us/supportRequest.png) 

盡量精確地填寫支援要求的每個區段。 當選擇服務，請確定您選取**Azure Active Directory 網域服務 （虛擬機器-網域控制站）** 程式要求路由傳送至適當的支援專業人員。

> [!IMPORTANT]
> 選取  **Azure Active Directory 目錄、 網域和物件**從服務清單中，與 Azure Active Directory 的網域驗證的相關說明。
>
> 

## <a name="feature-requests"></a>功能要求
您有了解我們如何改善 Azure Active Directory Domain Services 嗎？ 我們想要了解它。 請移至 Azure 的意見反應論壇，瀏覽至[網域服務](https://feedback.azure.com/forums/169401-azure-active-directory?category_id=160593)分類和分享您的想法。 產品群組的成員會檢閱這些要求。
