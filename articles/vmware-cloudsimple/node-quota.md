# <a name="cloudsimple-node-quota-limits"></a>CloudSimple 節點配額限制

四個節點時，可用來佈建，預設數量 CloudSimple 服務已啟用您的訂用帳戶。  您可以佈建任何[節點型別](cloudsimple-node.md)從 Azure 入口網站。  三個節點的相同 SKU 的最小值才能建立私人雲端。  如果您已佈建節點，可能會看到錯誤，當您嘗試佈建額外的節點中。

## <a name="quota-increase"></a>增加配額

您可以提交支援要求，以增加的節點配額。 服務作業小組會評估要求，並協助您增加節點配額。  當您開啟新的票證時，請選取下列選項：

* 問題類型：**技術**
* 訂用帳戶：**您的訂用帳戶識別碼**
* 服務類型：**VMware CloudSimple 方案**
* 問題類型：**專用的節點配額**
* 問題的子類型：**增加配額的專用節點**
* 主旨：**增加配額**

支援票證的詳細資訊，提供所需的節點和節點 SKU 數目。

您也可以連絡 Microsoft 客戶代表在[ azurevmwaresales@microsoft.com ](mailto:azurevmwaresales@microsoft.com)增加您的訂用帳戶上的節點配額。  您必須提供:

* 订阅 ID
* 節點的 SKU
* 您要為其要求增加配額的其他節點的數目
