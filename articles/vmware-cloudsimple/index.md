# <a name="azure-vmware-solution-by-cloudsimple"></a>由 CloudSimple 提供的 Azure VMware 解決方案

歡迎使用由 CloudSimple 提供的 Azure VMware 解決方案。 您可找到有關如何使用 CloudSimple 及如何在服務上啟用解決方案的詳細資訊。 搜尋文件或瀏覽目錄。

- 檢視[概觀](cloudsimple-vmware-solutions-overview.md)以深入了解由 CloudSimple 提供的 Azure VMware 解決方案。
- 選取 [CloudSimple 私人雲端概觀](cloudsimple-private-cloud.md)以深入了解 CloudSimple 私用雲端解決方案。
- 選取[節點](cloudsimple-node.md)以深入了解 CloudSimple 節點。
- 選取[初始化和佈建容量](quickstart-create-cloudsimple-service.md)以開始使用由 CloudSimple 提供的 Azure VMware 解決方案。

| 概念一覽     | 資源                    | 目的                                                                       | 管理性         |
|--------------------------|-----------------------------|-------------------------------------------------------------------------------|--------------------|
| **建立服務**     | CloudSimple 服務         | 設定 CloudSimple 服務邊緣網路                                    | Azure 入口網站       |
|                          | CloudSimple 節點            | 佈建容量以在 CloudSimple 服務中使用                          | Azure 入口網站       |
| **服務取用量**  | 私人雲端               | 建立 VMware 環境                                                    | CloudSimple 入口網站 |
|                          | VLAN/子網路                 | 為 VMware 耗用量建立基礎網路                               | CloudSimple 入口網站 |
|                          | 防火牆資料表              | 跨基礎網路設定南/北和東/西安全性規則 | CloudSimple 入口網站 |
|                          | 公用 IP 位址           | 取得 VMware 工作負載的公開位址                           | CloudSimple 入口網站 |
|                          | Azure 連線            | 在 Azure 中建立內部部署或虛擬網路的連線能力         | CloudSimple 入口網站 |
| **整合式管理**   | CloudSimple 虛擬機器 | 在 Azure 入口網站中管理 VMware 虛擬機器                                | Azure 入口網站       |
