# <a name="activity-management-overview"></a>活動管理概觀

CloudSimple 追蹤可能會影響您的私人雲端的運作狀況的所有活動。 這些包括警示、 事件、 工作和稽核活動的活動。 [活動頁面](https://docs.azure.cloudsimple.com/activity/)彙總所有目前的活動，可讓您向下切入其他詳細資料。

## <a name="events"></a>Events

事件追蹤 CloudSimple 入口網站上的使用者與系統活動。  事件會顯示與特定的資源和影響的嚴重性相關聯的活動。  您可以檢視從 CloudSimple 入口網站的事件。

## <a name="alerts"></a>警示

警示是在 CloudSimple 環境上任何重要活動的通知。  事件，會影響帳單或使用者的存取，會顯示成警示。  您可以確認 CloudSimple 入口網站中引發的警示。

## <a name="tasks"></a>工作

工作會追蹤任何使用者作業，需要超過 30 秒才能完成。  您可以監視從 CloudSimple 入口網站工作的進度。  完成的工作顯示工作完成所花費的總時間。

## <a name="audit"></a>稽核

稽核記錄的使用者執行的追蹤作業。  記錄檔包含使用者提供作業的參數。  您可以使用稽核記錄來監視使用者活動的所有使用者。
