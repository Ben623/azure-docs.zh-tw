# <a name="activity-management-overview"></a>活動管理總覽

CloudSimple 會追蹤可能會影響私人雲端運作的所有活動。 這些活動包括警示、事件、工作和 audit 活動。 [活動頁面](https://docs.azure.cloudsimple.com/activity/)會摘要所有目前的活動, 並可讓您向下切入以取得其他詳細資料。

## <a name="events"></a>Events

事件會在 CloudSimple 入口網站上追蹤使用者和系統活動。  事件會顯示與特定資源相關聯的活動, 以及影響的嚴重性。  您可以從 CloudSimple 入口網站查看事件。

## <a name="alerts"></a>警示

警示是您 CloudSimple 環境上任何重要活動的通知。  影響計費或使用者存取的事件會顯示為警示。  您可以從 CloudSimple 入口網站確認發出的警示。

## <a name="tasks"></a>工作

工作會追蹤任何使用者作業, 這需要30秒以上的時間才能完成。  您可以從 CloudSimple 入口網站監視工作的進度。  已完成的工作顯示工作完成所需的總時間。

## <a name="audit"></a>稽核

Audit logs 會追蹤使用者執行的作業。  記錄包含使用者為作業提供的參數。  您可以使用「審核記錄」來監視所有使用者的使用者活動。
