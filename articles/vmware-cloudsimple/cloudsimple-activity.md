# <a name="activity-management-overview"></a>活動管理總覽

AVS 會持續追蹤所有可能影響您的 AVS 私用雲端環境運作的活動。 活動包括警示、事件、工作和 audit 活動。 [活動頁面](monitor-activity.md)會摘要所有目前的活動，並可讓您向下切入以取得其他詳細資料。

## <a name="events"></a>活動

事件會追蹤 AVS 入口網站上的使用者和系統活動。 事件會顯示與特定資源相關聯的活動，以及影響的嚴重性。  您可以從 AVS 入口網站查看事件。

## <a name="alerts"></a>警示

警示是您的 AVS 環境中任何重要活動的通知。 影響計費或使用者存取的事件會顯示為警示。 您可以從 AVS 入口網站確認警示。

## <a name="tasks"></a>工作

工作會追蹤任何需要30秒以上才能完成的使用者作業。 您可以從 AVS 入口網站監視工作的進度。 若為已完成的工作，資訊會包含完成的總時間。

## <a name="audit"></a>稽核

Audit logs 會追蹤使用者作業。 Audit 記錄包含使用者為操作提供的參數。 您可以使用「審核記錄」來監視所有使用者的使用者活動。
