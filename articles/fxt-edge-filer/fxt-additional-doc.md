# <a name="additional-documentation"></a>其他文件 

此 docs.microsoft.com 網站以外的一些資源可協助您了解及使用您的 Microsoft Azure FXT 邊緣篩選混合式快取。

## <a name="avere-vfxt-for-azure-cache-documentation"></a>如需 Azure 快取文件的 Avere vFXT

[適用於 Azure 的 Avere vFXT](https://docs.microsoft.com/azure/avere-vfxt/)是類似的叢集快取產品在 Microsoft Azure 中使用虛擬節點。 它會透過 Azure 入口網站中，但 Azure FXT 邊緣篩選硬體快取會使用相同的控制面板設定軟體。 

## <a name="legacy-hardware-documentation"></a>舊版的硬體文件 

先前從 Avere 系統的產品中使用這項技術。 適用於這些較舊的產品的使用者文件位於[Avere 舊版文件網站](https://azure.github.io/Avere/)。 

具體來說，這些文件可能很有幫助的詳細資料：

* [叢集設定指南](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html)-包含常見的叢集設定和管理工作的概觀資訊，並有詳細的叢集的 [設定] 頁面的說明 

* [儀表板指南](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html)-說明控制台中的功能**儀表板** 索引標籤

* [FXT 叢集建立指南](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html)-叢集建立指南，從先前的產品
