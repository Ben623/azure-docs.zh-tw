# <a name="build-real-time-applications-with-azure-signalr"></a>使用 Azure SignalR 建立即時應用程式

[Azure SignalR](https://azure.microsoft.com/services/signalr-service/)可讓您輕鬆地將即時功能新增至應用程式。 這段即時功能可讓服務將內容更新推送至連線的行動應用程式。 因此，應用程式會更新，而不需要輪詢伺服器，或提交新的 HTTP 要求以進行更新。

## <a name="azure-signalr-features"></a>Azure SignalR 功能
- **廣泛的用戶端支援**-它適用于各種用戶端，例如 web 和行動瀏覽器、桌面應用程式、伺服器進程、IoT 裝置和遊戲主控台。
- SignalR 服務可讓**多個實例一起合作**，以擴充到數百萬個行動應用程式連線。 它也支援多個全球區域，以供分區化、高可用性或嚴重損壞修復之用。
- 為**不同的訊息模式提供豐富的 api** ，以將訊息傳送至特定的連線、所有連線，或屬於特定使用者或已放在任意群組中的連接子集。
