# <a name="build-real-time-applications-with-azure-signalr-service"></a>使用 Azure SignalR Service 建立即時應用程式

有了[Azure SignalR Service](https://azure.microsoft.com/services/signalr-service/)，您就可以輕鬆地將即時功能加入應用程式。 這段即時功能可讓服務將內容更新推送至連線的行動應用程式。 因此，應用程式會更新，而不需要輪詢伺服器或提交新的 HTTP 要求以進行更新。

## <a name="azure-signalr-service-features"></a>Azure SignalR Service 功能
- 有廣泛的用戶端支援，包括 web 和行動瀏覽器、桌面應用程式、伺服器進程、IoT 裝置和遊戲主控台。
- 可讓多個實例共同作業，以擴充到數百萬個行動應用程式連線。 它也支援多個全球區域，以供分區化、高可用性或嚴重損壞修復之用。
- 提供各種不同訊息模式的豐富 Api，以便將訊息傳送至特定的連線、所有連線，或屬於特定使用者或放在任意群組中的連接子集。
