# <a name="content-delivery-network-on-azure-with-azure-cdn-service"></a>Azure 上使用 Azure CDN 服務的內容傳遞網路
[AZURE CDN](https://azure.microsoft.com/services/cdn/)為開發人員提供一套全球解決方案，可將其內容快取到全球數百個存在點（PoP），快速傳遞高頻寬的內容，例如影片、影像或靜態檔案給使用者。 Azure CDN 也可以加速無法快取的動態內容，並可讓您減少載入時間、節省頻寬，並提高回應能力—無論您是開發或管理網站或行動應用程式，還是編碼和散佈串流媒體、遊戲軟體、固件更新或 IoT 端點。

## <a name="azure-cdn-features"></a>Azure CDN 功能
  - 與 Azure 服務**完美**搭配運作，包括 Web 應用程式、媒體服務、儲存體和雲端服務。
  - 輕鬆**處理流量尖峰和負載過重**。
  - **與 Akamai 和 Verizon 整合**，以建立及量身打造內容傳遞策略。
  - 動態網站加速。
  - 使用快取**規則**控制 Azure CDN 快取行為。
  - HTTPS**自訂網域支援**。
  - **Azure 診斷記錄**以查看核心分析，並將其匯出至 Blob 儲存體、事件中樞或 Azure 監視器。
  - 藉由**壓縮檔**案來改善效能。
  - 依據國家/地區限制 Azure CDN 內容的**地區篩選**。
