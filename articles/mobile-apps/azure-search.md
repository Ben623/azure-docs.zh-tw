# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>具備 AI 能力的雲端服務與 Azure 認知搜尋
[Azure 認知搜尋](https://azure.microsoft.com/services/search/)是一個搜尋即服務雲端解決方案，可為開發人員提供 api 和工具，以在 web、行動和企業應用程式中，為私人、異類內容新增豐富的搜尋體驗。 您的自訂程式碼會叫用資料擷取 (索引編制) 來建立和載入索引。 另一方面，應用程式的程式碼會發出查詢要求和處理回應。 搜尋體驗是在您的用戶端使用 Azure 認知搜尋的功能來定義，其中查詢會透過您建立、擁有並儲存在服務中的保存索引執行。

## <a name="azure-cognitive-search-features"></a>Azure 認知搜尋功能
- **自由格式文字搜尋**：提供全文檢索搜尋、簡單查詢語法和 Lucene 查詢語法。
- **簡單評分設定檔**：模型相關性是檔本身中的值函數。
- **地理搜尋功能**：根據搜尋結果與實體位置的鄰近程度來探索資料。
- **使用者體驗功能**：提供自動完成、搜尋建議、排序和分頁。
- **認知搜尋**：可以套用至索引管線，從原始內容中解壓縮影像和文字分析的文字資訊。
- **知識存放區：儲存**在索引編制期間建立的擴充。
- **資料來源**：當您連線至 Azure SQL Database、Azure Cosmos DB 和 Azure Blob 儲存體時，請將主要資料存放區中可搜尋的內容解壓縮。
- 匯**入資料嚮導**：設定索引子。 
- **索引設計**工具：代表索引。 
- **搜尋瀏覽器**：測試查詢並改善評分設定檔。

尋找 Azure 認知搜尋[功能](/azure/search/search-what-is-azure-search#feature-descriptions)的詳細描述。
