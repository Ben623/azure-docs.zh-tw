# <a name="mainframe-workloads-supported-on-azure"></a>在 Azure 上支援的大型主機工作負載

這裡列出的解決方案，可協助您將您的大型主機工作負載移轉至 Microsoft Azure。 某些工作負載比較容易移轉。 取決於舊版的系統軟體的其他工作負載可以重新裝載。 

與大型主機模擬和服務的更多協助，請參閱[Azure 大型主機移轉中心](https://azure.microsoft.com/migration/mainframe/)。

## <a name="migrate-mainframe-closer-to-azure"></a>將更接近的大型主機移轉至 Azure

- [Micro 焦點 Visual COBOL](https://www.microfocus.com/products/visual-cobol/)開發和整合的工具。
- [Micro 焦點 PL / 我](https://www.microfocus.com/campaign/download/pli-modernization/)舊版編譯器為.NET 平台，支援大型主機 PL / 我的語法、 資料類型和行為。
- [Micro Focus Enterprise Server](https://www.microfocus.com/products/enterprise-suite/enterprise-server/)大型主機整合平台。
- [Fujitsu NetCOBOL](https://www.fujitsu.com/global/products/software/developer-tool/netcobol/)開發和整合的工具。
- [NTT 資料企業 COBOL](https://www.nttdataservices.com/en-ie/services/application-modernization-services)開發和整合的工具。
- [開啟 NTT PL / 我](https://www.nttdataservices.com/en-ie/services/application-modernization-services)舊版編譯器為.NET 平台，支援大型主機 PL / 我的語法、 資料類型和行為。
- [Raincode COBOL 編譯器](https://www.raincode.com/products/cobol/)開發和整合的工具。
- [Raincode PL / 我編譯器](https://www.raincode.com/products/pli/)適用於.NET 平台支援大型主機 PL / 我的語法、 資料類型和行為。
- [Raincode ASM370 編譯器](https://www.raincode.com/technical-landscape/asm370/)的大型主機組合器 370 和 HLASM 語法。
- [適用於.NET 的 ASNA Visual RPG](https://asna.com/us/products/visual-rpg)編譯器為.NET Framework，從 Visual Studio 外掛程式。
- [現代系統 CTU （COBOL-通用）](https://modernsystems.com/automatic-cobol-to-java-conversion/)開發和整合的工具。

## <a name="deploy-an-emulation-environment-for-online-and-batch-processing"></a>部署的模擬環境的線上和批次方式處理

- [Micro Focus Enterprise Server](https://www.microfocus.com/products/enterprise-suite/enterprise-server/)大型主機整合平台。
- [NTT 資料大型主機重新裝載開發環境](https://us.nttdata.com/en/-/media/assets/white-paper/apps-mainframe-re-hosting-development-environment-whitepaper.pdf)原生交易處理環境。
- [NTT 資料批次處理環境](https://us.nttdata.com/en/-/media/assets/white-paper/apps-mainframe-re-hosting-development-environment-whitepaper.pdf)(BPE) 包括 JCL 交易功能。
- [Raincode CICS](https://www.raincode.com/technical-landscape/cics/) .NET 和 Azure 平台的模擬器。
- [Raincode JCL](https://www.raincode.com/products/jcl/) plug-compatible JCL 解譯器。

## <a name="code-conversion"></a>程式碼轉換

- [Asysco](https://www.asysco.com/azure-cloud/)來源的程式碼轉換。
- [LzLabs 軟體定義大型主機](https://www.lzlabs.com/)受管理的軟體容器，將大型主機應用程式移轉至 Linux 電腦或公用、 私人和混合式雲端環境。
- [藍光時代](https://www.bluage.com/)數位傳統商務應用程式和資料庫的工具。
