# <a name="micro-focus-devtest-environments-on-azure"></a>在 Azure 上的微焦點開發/測試環境

開始在 Azure 上使用的開發與測試環境是適合用來提升現代化的成果。 相依於舊版的系統，例如 IBM CICS （客戶資訊控制系統） 和 IMS （資訊管理系統） 的大型主機應用程式都可以使用協力廠商解決方案在 Azure 上重新裝載。

Micro Focus 具有廠與 Microsoft 合作提供吸引人的大型主機移轉與重新裝載的工具。 您可以執行跨開發、 測試及生產案例的大型電腦應用程式，在 Azure 和 Microsoft Windows 平台上。

Micro Focus 提供重新裝載的開發和測試重新裝載解決方案擴充現有的 COBOL 和 PL / 我商務應用程式至 Azure。
